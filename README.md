# Gallery app 

The app is built using React & node.js

```sh
git clone https://github.com/zacharychoi-ce/gallery-app-react.git
cd gallery-app-react && npm i
```

## Stretch
- Move database to AWS RDS - mySQL, server-side to AWS Lambda and API Gateway
- Update Posts
- Styling

## Running the app

```sh
npm run dev
```

On browser, run:
```sh
http://localhost:3000
```
